# Emosson

Emosson est un site de prévention. Il s'inscrit dans un contexte de prévention des risques naturels et vise à proposer un outil web permettant d’informer les utilisateurs sur l’exposition d’un territoire aux risques d’inondations.

## Projet Emosson 

Le site web est composé de trois pages : une page d’accueil pour présenter le sujet, une page questionnaire destinée à recueillir un maximum d’informations sur le lieu et le type de logement, et enfin une page de résultats qui proposera des n score de vulnérabilité et des recommandations pour protéger le bien avec la possibilité de télécharger le bilan sous forme de fichier PDF.
